# Cover Songs Workshop

This folder is a repertoire and production workspace, not an archive of copied recordings or lyrics.

## Safe workflow

1. Confirm the composition's status in the release territory. “Traditional” is not automatically public domain.
2. Record a new performance. Do not sample another artist's master without permission.
3. Credit the known writers and arrangers accurately.
4. For a copyrighted composition, use the cover-song licensing workflow offered by the chosen distributor and follow its territory rules.
5. Obtain direct written permission for adaptations, translated lyrics, samples, medleys, or changed fundamental lyrics/melody.
6. Keep permission receipts and release notes outside the public site if they contain personal data.

`repertoire.csv` is deliberately plain UTF-8 CSV so it works in a text editor or spreadsheet.
